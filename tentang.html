Yuk Mengenal Lebih Dekat Komunitas Kami

Siapa Kami

Kami adalah sekelompok anak muda pecinta kereta api yang peduli akan keselamatan dan ketertiban di perlintasan kereta api. Komunitas pecinta kereta api yang eksis sejak 2009 ini kini beranggotakan sekitar 180 orang lebih, dengan rata-rata usia 15 hingga 25 tahun.

Tujuan

Menghimpun dan menyatukan anggota, Mewujudkan dan meningkatkan kualitas hidup & profesionalitas anggota, Mengajak peran serta yang lebih profesional Operator maupun Regulator sebagai perwujudan dari Pelayanan Masyarakat.

Tahun Didirikan

ESI didirikan oleh Egief Del Haris, Desya Nur Perdana, Armiya Farhana, Budi Susilo, Agus Riyadi, Luqman Supriyatno pada tanggal 5 Juli 2009 di Jatinegara.

Kegiatan kami

Disiplin Perlintasan

Kegiatan #DisiplinPerlintasan merupakan kegiatan edukasi di Perlintasan Sebidang dan Terbuka untuk siapapun baik dari masyarakat dan komunitas pecinta kereta api lainnya dengan menerapkan protokol kesehatan. Jadwal Kegiatan #DisiplinPerlintasan diadakan setiap hari Jum’at mulai dari jam 15.00 di Pos Jaga Perlintasan.

Bebersih Petak

Bebersih Petak merupakan sebuah aksi teladan dalam bentuk gerakan pungut sampah dan edukasi pengelolaan sampah di sepanjang lingkungan bantaran rel kereta api. Selain itu, sepanjang kegiatan #BebersihPetak, kegiatan ini akan didampingi oleh rekan-rekan @bandunggoodguide yang akan melakukan storytelling di beberapa checkpoint yang dilalui nantinya. Jadi, Peserta bisa membersihkan lingkungan sambil dapat pengetahuan. Tentu saja ini akan menambah keseruan Bebersih Petak kali ini.

Bahaya menerobos Palang Pintu Kereta

Palang pintu pada perlintasan kereta api tujuannya untuk mencegah pengguna jalan terutama pengendara menerobos rel karena ada kereta yang hendak melintas. Tujuannya jelas untuk keselamatan pengguna jalan dan kereta api yang akan melintas dengan cara menurunkan palang pintu. Selain palang pintu, di perlintasan kereta api juga ada peringatan melalui pengeras suara.

Hanya saja fungsi palang pintu untuk mencegah terjadinya kecelekaan sering kali diabaikan pengguna jalan. Kita sering melihat sejumlah pengendara motor nekat melintasi rel kendati palang pintu kereta api sudah diturunkan petugas. Sehingga sering terjadi pengendara motor atau mobil meninggal karena menerobos palang pintu kereta api. Tertabraknya pengguna jalan ini jelas merugikan diri sendiri, selain itu juga melanggar aturan. Seperti dikutip dari Kabaroto.com dan Autofun.co.id, menerobos palang pintu kereta api termasuk perbuatan melawan hukum, hal ini diatur dalam Pasal 114 Undang-Undang Nomor 22 Tahun 2009 tentang Lalu Lintas dan Angkutan Jalan (UULLAJ), yang berbunyi:

“Pada perlintasan sebidang antara jalur kereta api dan jalan, pengemudi kendaraan wajib:
a. Berhenti ketika sinyal sudah berbunyi, palang pintu kereta api sudah mulai ditutup dan/atau ada isyarat lain,
b. Mendahulukan kereta api
c. Memberikan hak utama kepada kendaraan yang lebih dahulu melintasi rel.”

A turannya jelas, bahwa ketika sinyal berbunyi, palang pintu kereta api sudah mulai ditutup atau ada isyarat lain pengemdui kendaraan wajib mendahulukan kereta api.

Kegiatan kami

a. Meningkatkan kualitas komunikasi anggota dengan cara mempertinggi mutu informasi dan komunikasi secara selektif

b. Mengakomodasi kegiatan-kegiatan sesama anggota untuk menambah dan memenuhi kebutuhan sendiri serta kegiatan lain yang sah dan bermanfaat serta tidak bertentangan dengan AD/ART.

c. Mempertebal rasa ingin membantu sesama anggota didalam peningkatan kualitas hidup dan profesionalisme

d. Mewujudkan rasa kesetiakawanan sosial antara sesama anggota dan masyarakat pada umumnya dengan kegiatan nyata.

e. Melakukan pertemuan-pertemuan rutin dengan pihak Operator maupun Regulator yang berfungsi sebagai sarana menyampaikan aspirasi maupun saling bertukar pikiran.
